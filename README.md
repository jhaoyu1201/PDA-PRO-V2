
# 透視繪圖助手 Pro (Perspective Art Studio)

這是一款專為創作者設計的專業級透視線輔助網頁工具。旨在幫助畫師在處理複雜透視（如建築、大場景）時，能快速定位消失點並生成乾淨的透視參考線。

## ✨ 主要功能

- **多點透視支援**：靈活設置 1 點、2 點、3 點甚至 4 點透視。
- **透視反推偵測 (Guide Mode)**：可透過在畫面中畫出兩條特徵線段，自動計算交點並新增消失點。
- **即時互動操作**：
  - `Alt + 滑鼠左鍵`：平移畫布空間。
  - `滑鼠滾輪`：縮放畫布。
  - `Ctrl + V`：直接從剪貼簿貼上參考圖片。
  - `Delete / Backspace`：快速刪除選中的消失點。
- **智慧對齊 (Snapping)**：拖曳點位時自動對齊其他消失點的水平或垂直座標。
- **復原功能 (Undo)**：支援 `Ctrl + Z` 歷史紀錄回溯。
- **專業輸出**：輸出一張與原圖尺寸完全一致的透明底 PNG 透視網格。

## 🛠 技術棧

- **框架**: React 19 (TypeScript)
- **樣式**: Tailwind CSS
- **渲染**: HTML5 Canvas API
- **部署支援**: Vite / Vercel

## 🚀 部署指南

### 在本地開發
1. 安裝依賴：
   ```bash
   npm install
   ```
2. 啟動開發伺服器：
   ```bash
   npm run dev
   ```

### 部署到 Vercel
1. 將此專案推送到您的 GitHub 倉庫。
2. 在 [Vercel](https://vercel.com/) 點擊 "Add New Project"。
3. 連結您的 GitHub 倉庫。
4. 框架預設選擇 **Vite**，點擊 **Deploy** 即可完成。

## ⌨️ 快捷鍵說明

| 快捷鍵 | 動作 |
| :--- | :--- |
| **Ctrl + V** | 貼上圖片 |
| **Ctrl + Z** | 復原上一步 |
| **Alt + 左鍵** | 平移畫布 |
| **Delete** | 刪除選中消失點 |
| **Esc** | 關閉右鍵選單 / 退出偵測模式 |
